Everything under the hood

YourBot is more than a link downloader — it's a full media toolkit inside Telegram. Here's a complete tour of what it can do.

🎬 Video downloads

Send any video link and the bot downloads it straight into the chat — no redirects, no third-party sites. The engine behind it is yt-dlp, the most actively maintained downloader out there, so newly supported sites and fixes land all the time.

  • 1000+ supported platforms — YouTube, Instagram, TikTok, X/Twitter, Facebook, VK, Vimeo, Twitch, Rutube, Dailymotion, SoundCloud and many more
  • Up to 10 links in one message when quality-selection mode is on (one per line or separated by spaces)
  • Live streams can be captured too
  • Reply to any video with text to change its caption
  • Full MediaInfo data on request — codec, bitrate, resolution, duration
ℹ️

Not sure if a site works? Just send the link — the bot will tell you. DRM-protected platforms (Netflix, Disney+, etc.) are intentionally blocked.

🎵 Audio extraction

Turn any video into an audio file with a single command. The bot extracts the best available audio track and sends it as a playable Telegram audio message with title and thumbnail.

chat
/audio https://www.youtube.com/watch?v=...

Audio mode also understands playlist ranges, so you can rip a whole album in one go.

📋 Playlists & ranges

Downloading one-by-one is so 2019. Append a range to any playlist URL, or use the /vid command:

PatternWhat it does
URL*1*5Downloads items 1 to 5
URL*-1*-5Downloads the last 5 items
URL*3*3Only the 3rd item
/vid 3-7 URLSame as URL*3*7
/vid -3-7 URLItems 3–7 counting from the end

Ranges work on YouTube playlists, TikTok profiles, Instagram stories & highlights, VK video pages, Rutube channels, Twitch clips, Vimeo groups and more — up to 50 items per run (100 in groups).

🖼 Images & galleries

The /img command pulls full-resolution images, albums and entire galleries using gallery-dl. Files arrive as neat 10-item media albums.

chat
/img https://www.instagram.com/p/C8x...
/img 11-20 https://example.com/album
/img 11- https://example.com/album

Works with Instagram, Pinterest, VK, Reddit, Twitter/X, TikTok, DeviantArt, ArtStation, Imgur, Tumblr, 4chan and many others. Limits: up to 1000 files per run in DM, 2000 in groups.

🎚 Quality & formats

You're in control of what comes out the other end:

  • /format ask — the bot shows available qualities as buttons for every link
  • /format 720p — lock a quality (144p up to 4K/8K where available)
  • /format best — always the best possible quality
  • /format id 134 — pick an exact format ID from /list
  • /list URL — see every available format for a link before downloading

💬 Subtitles

Download subtitles as separate files or embed them directly into the video. Auto-generated and auto-translated tracks are supported too.

chat
/subs          # subtitle menu
/subs en       # English subtitles
/subs auto     # auto-translated track
/subs off      # disable
⚠️

Embedding subtitles is heavy processing, so video+subs is limited to 720p, 500 MB and 1.5 hours for everyone.

🍪 Cookies & private content

Age-restricted YouTube video? Private Instagram post? Members-only content? Upload your browser cookies once and the bot downloads as you.

  • Send a cookie.txt file as a document — done
  • /save_as_cookie — paste cookie text right into the chat
  • /cookie [service] — download the bot's cookies for a service (youtube / tiktok / x / custom)
  • /cookie youtube 1 — pick a specific cookie source by index
  • /cookies_from_browser — extract cookies from your browser profile
  • /check_cookie — verify that your cookies still work
🔒

Cookies are stored per-user and only used for your own downloads. Never share your cookie file with strangers — it's a login key.

✂️ Splitting big files

Telegram bots can upload up to 8–16 GB, but sometimes you want smaller chunks. Set a part size and the bot splits the video with ffmpeg — no quality loss, no re-encoding.

chat
/split 500mb     # cut into 500 MB parts
/split 100mb-2000mb

🔗 Direct links

Don't want the file in Telegram, just the raw URL? /link returns direct download links you can paste into any download manager or browser.

chat
/link https://youtu.be/abc123          # best quality
/link 720 https://youtu.be/abc123      # specific quality
/link 1080p URL · /link 4k URL · /link 8k URL

🏷 Tags

Add hashtags after any URL and they'll be saved and shown in the file's caption — handy for sorting your Saved Messages into collections.

chat
https://tiktok.com/@user/video/123 #funny #cats
/tags     # view all your saved tags

🛡 Proxy support

Region-locked video? Route your downloads through a proxy: /proxy on and /proxy off. The bot supports HTTP, HTTPS, SOCKS4 and SOCKS5 proxies, and can rotate between multiple proxy servers automatically.

🔞 NSFW mode

Adult content is clearly marked, blurred by default, and logged separately. In DMs it's a paid feature (1 ⭐ = $0.02) — inside groups it's free with full metadata. Toggle the spoiler blur any time with /nsfw on / /nsfw off.

⚡ Smart cache

Every downloaded file is cached by URL. If you (or someone else) request the same link again, the bot re-sends the stored file instantly instead of re-downloading it. Use /uncache URL to force a fresh copy, and /usage to see your download history.

👥 Group mode

Add YourBot to a group and it gets noticeably better for everyone inside:

  • All limits doubled (see the comparison on the home page)
  • Media auto-sorted into topics, replies in the topic it was called from
  • Auto-pinned progress message
  • /img delivers 10-item albums
  • NSFW content free, with full metadata
  • No forced channel subscription
  • No logging or caching of group content (configurable)

Send /add_bot_to_group in the bot for the full rundown.
