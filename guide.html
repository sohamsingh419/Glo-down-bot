The complete user guide

From your very first link to playlists, cookies and subtitles — everything explained with copy-paste examples.

1. Quick start

Open @GlobalXdownloaderBot and press Start. If the bot asks you to join our channel @Globall_X first — join, then resend your link. That's the whole onboarding.

⚠️

Don't leave the channel afterwards — the bot checks membership and leaving will get you restricted.

2. Your first download

Copy a link to any video — a YouTube clip, an Instagram reel, a TikTok — and paste it into the chat. The bot detects the site, fetches the media and uploads it back to you. That's it.

chat
https://www.youtube.com/watch?v=dQw4w9WgXcQ

Links must start with https:// or http:// — plain text without a scheme won't be recognised.

3. Choosing quality

By default the bot picks a sensible quality automatically. To take control, use /format:

CommandBehaviour
/format askShows a quality menu with buttons for every link you send
/format 720pAlways downloads 720p (also 144p … 1080p, 4k, 8k)
/format bestAlways the highest available quality
/format id 134Pins an exact format ID (find IDs with /list URL)

All quality settings live in /settings too, if you prefer buttons over typing.

4. Multiple links at once

In quality-selection mode (/format ask) you can send up to 10 URLs in a single message — each on its own line or separated by spaces. The bot queues them all and picks up where it left off if anything interrupts.

5. Playlists & ranges

Append a range to the end of any playlist URL: URL*start*end. Negative numbers count from the end.

examples — copy & send
# First 5 videos of a YouTube playlist
https://youtu.be/playlist?list=PL...*1*5

# Only the 3rd video
https://youtu.be/playlist?list=PL...*3*3

# First 10 videos from a TikTok profile (needs cookies)
https://www.tiktok.com/@USERNAME*1*10

# First 3 Instagram stories (needs cookies)
https://www.instagram.com/stories/USERNAME*1*3

# First 3 videos from a VK page
https://vkvideo.ru/@PAGE_NAME*1*3

# Videos 2 to 4 from a Rutube channel
https://rutube.ru/channel/CHANNEL_ID/videos*2*4

# First 3 clips from a Twitch channel
https://www.twitch.tv/USERNAME/clips*1*3

Prefer a command? /vid 3-7 URL does the same as URL*3*7, and /vid -3-7 URL counts from the end. Works with /audio too. Limit: 50 items per run (100 in groups).

6. Audio only

Prefix any link with /audio and you'll get just the sound — neatly tagged with title and cover art where available.

chat
/audio https://www.youtube.com/watch?v=...
/audio https://soundcloud.com/artist/track

7. Subtitles

Open /subs and pick a language, or set it directly:

chat
/subs en      # English subtitles
/subs hi      # Hindi subtitles
/subs auto    # auto-generated / auto-translated
/subs off     # no subtitles
⚠️

Burning subtitles into video takes real processing power, so video+subs is capped at 720p · 500 MB · 1.5 hours.

8. Images & albums

/img downloads images, albums and whole galleries in full resolution:

examples
/img https://vk.com/wall-160916577_408508
/img https://x.com/username/status/1234567890123456789
/img https://imgur.com/a/abc123
/img 11-20 https://example.com/album   # items 11..20 only
/img 11- https://example.com/album      # from 11 to the end

Instagram, Pinterest, VK, Reddit, Twitter/X, TikTok, DeviantArt, ArtStation, Imgur, Tumblr and dozens more. Up to 1000 files per run — the bot stops automatically if a gallery goes quiet for 5 minutes.

9. Cookies — private & age-restricted content

Some content requires you to be logged in: age-restricted YouTube videos, private Instagram accounts, TikTok profiles, Instagram stories. The solution is cookies — your browser's login session, exported as a file.

Option A — send a cookie file (easiest)

  • Install a cookie-export extension in your browser (e.g. "Get cookies.txt LOCALLY")
  • Export cookies for the site you need and save the file as cookie.txt
  • Send that file to the bot as a document — done

Option B — paste as text

Send /save_as_cookie followed by the cookie content in Netscape format:

chat
/save_as_cookie
# Netscape HTTP Cookie File
.youtube.com  TRUE  /  FALSE  111  ST-xxxxx  session_logininfo=AAA
.youtube.com  TRUE  /  FALSE  222  ST-xxxxx  session_logininfo=BBB

Keep them healthy

  • /check_cookie — verify your cookies still work
  • /cookie youtube — download the bot's shared cookies for a service
  • /cookie youtube 2 — pick a backup cookie source by index
🔒

Your cookie file is a login key. The bot stores it per-user for your downloads only — never share it in public chats.

10. Direct links instead of files

Sometimes you want the raw download URL — for a download manager, another device, or sharing. /link gives you exactly that:

chat
/link https://youtu.be/abc123         # best quality
/link 720 https://youtu.be/abc123     # specific quality

11. Tags & captions

Add hashtags after any URL to label the download — they're saved and shown in the caption, which makes Telegram's Saved Messages search genuinely useful:

chat
https://tiktok.com/@user/video/123 #recipes #italian
/tags     # list everything you've tagged

You can also reply to any video with text to rewrite its caption.

12. Cleaning up

Want a fresh start? /clean removes your media files, and you can target specific data:

chat
/clean            # media files only
/clean all        # everything
/clean cookies    # just cookies
/clean tags       # just tags
/clean format     # quality settings
/clean sub        # subtitle settings

And if a cached file looks stale, /uncache URL forces a fresh download next time.
